[wow]https://wod.wowhead.com/spell=49998[/wow]

[en]

Usage

Copy a Wowhead.com item/spell/archievement/... link into your text to display the tooltip.

Use advanced parameters to adjust the presentation.


The tooltip works with all languages supported by Wowhead:

https://www.wowhead.com/item=28288 = Displays an english tooltip
https://de.wowhead.com/item=28288 = Displays an german tooltip
https://fr.wowhead.com/item=28288 = Displays an french tooltip

Examples: 

[wow]https://www.wowhead.com/item=112920/korvens-crimson-crescent[/wow]
[wow=bonus=450&lvl=52&spec=263]https://www.wowhead.com/item=112920[/wow]
[wow]https://www.wowhead.com/item=112920/korvens-crimson-crescent&bonus=450&lvl=53&spec=263[/wow]
[wow]https://www.wowhead.com/item=28288[/wow]
[wow]https://wod.wowhead.com/spell=49998[/wow]
[wow=gems=23121&ench=2647&pcs=25695:25696:25697]https://www.wowhead.com/item=25697[/wow]
[wow=upgd=4]https://www.wowhead.com/item=105547[/wow]
[wow=upgd=4]https://de.wowhead.com/item=105547[/wow]
[wow]https://classic.wowhead.com/item=19019/thunderfury-blessed-blade-of-the-windseeker[/wow]

Change tooltip settings

Create a copy of __wowheadBBCodeInclude.tpl and change the values as you like.

Don't forget to select the template group in your style!

[/en]
